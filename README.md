# RoundNoye.github.io

A personal bio website built with HTML, CSS, and JavaScript. This site serves as an online portfolio and personal description.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Interactive Elements**: Smooth scrolling, typing effects, and hover animations
- **Dark Mode**: Toggle between light and dark themes
- **Scroll Progress**: Visual indicator showing scroll progress
- **Sections**: About, Skills, Projects, and Contact information

## Customization

To personalize this website:

1. **Update Personal Information**:
   - Edit `index.html` to change your name, bio, skills, and contact details
   - Replace placeholder project descriptions with your actual projects
   - Update social media links in the contact section

2. **Add Your Photo**:
   - Replace the placeholder image in the about section with your actual photo
   - Update the CSS in `style.css` to adjust image sizing if needed

3. **Customize Styling**:
   - Modify colors, fonts, and layout in `style.css`
   - The design uses CSS Grid and Flexbox for responsive layouts

4. **Add More Sections**:
   - Copy existing section structures in `index.html`
   - Add corresponding styles in `style.css`
   - Update navigation if needed

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript (ES6+)

## Deployment

This site is automatically deployed via GitHub Pages. Any changes pushed to the `main` branch will be live at `https://roundnoye.github.io`.

## Local Development

To run locally:
1. Clone the repository
2. Open `index.html` in your browser
3. For live reloading during development, use a local server (e.g., `python -m http.server` or VS Code Live Server extension)
